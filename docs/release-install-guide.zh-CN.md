# 安装说明

请前往 [GitHub Releases](https://github.com/coderhsh/nrm-desktop/releases) 页面下载安装包。

## macOS

**`.dmg`（推荐 macOS 用户使用）**：下载后双击打开，将应用图标拖入「应用程序」文件夹即可完成安装。

> **说明**：默认 Release 提供 Apple Silicon（M1 / M2 / M3 / M4）版本。如需 Intel Mac 版本，可通过 CI 手动构建获取。

## Windows

1. **`setup.exe`（推荐）**：常见安装程序，会自动创建开始菜单和桌面快捷方式。
2. **`.zip`（便携版）**：解压后直接运行 `nrm-desktop.exe`，无需安装。
3. **`.msi`**：面向企业 IT 管理员，支持静默安装与批量部署。

## 系统要求

- **macOS**：macOS 10.15 或更高版本（Apple Silicon）
- **Windows**：Windows 10 / 11 x64，需安装 [WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)
